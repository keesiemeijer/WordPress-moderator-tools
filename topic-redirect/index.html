WordPress topic redirect

A Greasemonkey user script that forces https, adds view=all to topic urls and redirects if needed. Tested in Firefox and Chrome.

Firefox install

Install Greasemonkey (If you haven't done so already).

Click to install the: redirect user script

Chrome install

  1. Install the Tampermonkey Chrome extension to manage your user scripts. This extension allows scripts to be added outside the Chrome Web Store directly.
  2. Click this Chrome Greasemonkey user script link to have Tampermonkey install it.
